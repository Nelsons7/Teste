Aplicações em Array



Exemplo 1

Escolha 30 frases. Coloque as frases em um array (cuidado com as aspas). Crie um código que a cada visita a página HTML, uma frase diferente é exibida. Bônus: Crie um app com um botão o qual quando pressionado, uma nova frase é exibida (para que não seja necessário dar refresh na página para exibir uma nova frase).



Exemplo 2

Vá ao Google Imagens. Escolha 10 imagens de natureza. Copie os endereços URL das 10 imagens e coloque em um array. Crie um código que a cada visita a página HTML, uma imagem diferente é exibida. Bônus: Crie um app com um botão o qual quando pressionado, uma nova imagem é exibida (para que não seja necessário dar refresh na página para exibir uma nova imagem).
